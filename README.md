## Overview
This is a simple command-line calculator written in C++.  
It supports basic arithmetic operations: **addition (+), subtraction (-), multiplication (*), division (/), and modulus (%)**.  
The program runs in a loop, allowing the user to evaluate multiple expressions until they choose to exit.
